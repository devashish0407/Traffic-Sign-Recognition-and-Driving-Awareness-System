System Architecture & Pipeline Flow

Trace the workflow of the Traffic Lens computer vision threat detection engine

1

Telemetry Feed Ingestion

Select between live hardware camera streams or local video recording files (supporting up to 200 MB sizes) in the dashboard panel.

2

Dynamic Spatial Scaling

Normalizes input frames to a uniform 640x480 resolution. This optimizes model tensor execution and keeps processing speeds high.

3

YOLOv8 Boundary Detection

A deep neural network detects sign boundary boxes in real-time, matching objects with a confidence score above a 0.50 threshold.

4

Secondary Classification Crop

Crops boundary boxes and runs a fine-grained secondary classifier model to identify speed limits, stops, yields, and warnings.

5

Hazard & Alert Evaluation

Calculates caution flags and emits alert levels. Applies temporal smoothing and sign cooldown parameters to avoid message spam.

6

Telemetry Logs & Analytics

Feeds analytics trends to Chart.js charts and organizes real-time logs, which are exportable to downloadable CSV reports.
